🏆 Sports Hub

A fun and interactive web project featuring **mini games (Football & Hand Cricket)** and a **contact form with validation**. Built using **HTML, CSS, and JavaScript**, this project demonstrates DOM manipulation, game logic, animations, and local storage.

---

## 🚀 Live Features

### ⚽ Football Penalty Game

* Choose direction: Left / Center / Right
* Random goalkeeper AI movement
* Goal / Save detection system
* Score tracking:

  * Goals
  * Attempts
* Flash animations for goals and saves
* Match history stored in browser (localStorage)

---

### 🏏 Hand Cricket Game

* Choose number from 1 to 6
* Computer randomly selects number
* Rules:

  * Same number = OUT
  * Different number = runs added
* Tracks:

  * Total runs
  * Game status (Batting / Out)
* Saves last scores using localStorage

---

### 📩 Contact Form

* Input fields:

  * Name
  * Email
  * Message
* Validation rules:

  * No empty fields
  * Email must contain "@"
* Displays:

  * Error messages
  * Success message on valid submission

## 🎮 How to Run

1. Download or clone the project
2. Open folder in VS Code or any editor
3. Run using browser:

   * `index.html` → Home page
   * `game.html` → Play games
   * `contact.html` → Contact form

No server required (pure frontend project)
